# 90 天执行看板

## 目标

90 天内验证“高手思维拆解知识库 + 社群文字答疑 + 个人方法论诊断”是否能形成正向现金流。

## 第 1-2 周：知识库 MVP

| 任务 | 状态 | 验收标准 |
|---|---|---|
| 搭建飞书/Notion 首页 | 未开始 | 6 个入口完整 |
| 完成 10 个高手案例 | 未开始 | 每个案例使用统一模板 |
| 完成 3 个拆解方法 | 未开始 | 五层拆解法、四问法、三步法 |
| 完成 4 个工具模板 | 未开始 | 画布、观点卡、选题卡、转化模板 |
| 完成 1 个公开样章 | 未开始 | 可用于私信发放 |
| 完成销售页 | 未开始 | 可直接收款前沟通 |
| 建立社群规则 | 未开始 | 明确非即时答疑边界 |

## 第 3-4 周：预热与首发

| 任务 | 状态 | 验收标准 |
|---|---|---|
| 发布 15 条视频号内容 | 未开始 | 每条有转化动作 |
| 收集样章领取用户 | 未开始 | 记录领取人数 |
| 私域跟进咨询 | 未开始 | 记录咨询到购买转化 |
| 完成首批成交 | 未开始 | 目标 20 单 |
| 收集购买理由 | 未开始 | 至少 10 条有效反馈 |
| 收集未购买理由 | 未开始 | 至少 10 条有效反馈 |

## 第 2 个月：优化转化

| 任务 | 状态 | 验收标准 |

| 整理社群高频问题 | 未开始 | 至少 20 条 FAQ |
| 更新知识库 2 轮 | 未开始 | 每轮有更新说明 |
| 做 3 个内容应用示例 | 未开始 | 视频号/公众号/小红书各有示例 |
| 优化销售页 | 未开始 | 增加 FAQ 和用户反馈 |
| 复盘内容转化 | 未开始 | 找出前 3 条有效选题 |

## 第 3 个月：升级诊断

| 任务 | 状态 | 验收标准 |
|---|---|---|
| 设计诊断申请表 | 未开始 | 能筛选合适用户 |
| 邀请 5-10 个用户 | 未开始 | 来自知识库购买者 |
| 成交 3 单诊断 | 未开始 | 验证升级路径 |
| 形成 2-3 个案例 | 未开始 | 可匿名公开 |
| 调整诊断定价 | 未开始 | 判断是否升到 1999/2999 |

## 每周复盘问题

1. 本周卖了多少单？
2. 用户为什么买？
3. 用户为什么没买？
4. 哪条内容带来最多私信？
5. 社群里最多的问题是什么？
6. 哪些问题可以沉淀进知识库？
7. 答疑有没有超过可控边界？
8. 有没有用户适合转诊断？

## 90 天是否继续放量的判断

继续冲：

- 知识库卖出 50 单以上。
- 退款率低于 5%。
- 社群答疑每周不超过 4 小时。
- 至少成交 3 单诊断。
- 视频号能持续带来私信或样章领取。

黄灯模式：

- 知识库卖得动，但用户不会用。
- 社群答疑时间开始失控。
- 有咨询，但成交理由不清楚。
- 内容数据不错，但私域转化弱。

立即收缩：

- 用户大量投诉交付不清楚。
- 退款率超过 10%。
- 你明显不想再做答疑。
- 视频号连续 30 天无法带来有效线索。
