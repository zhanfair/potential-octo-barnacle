# 宫崎骏第一条视频结构稿修订意见

## 修订结论

当前稿件结论：

```text
小修后可进入资料补强。
```

它不需要推倒重写，核心命题已经成立：

```text
宫崎骏不是等灵感的人，他是用动作把答案逼出来的人。
```

但现在的问题是：结构上有一处证据错位，开头的观看钩子还可以更狠，结尾的“普通人可借走动作”需要更早埋入。

## 多人圆桌审稿

### 稿件任务判断

- 目标格式：5-7 分钟高手思维拆解长视频结构稿。
- 目标受众：知识型内容创业者、个人 IP 起步者、输入很多但输出不稳定的人。
- 核心主张：高手不是先想清楚再开始，而是在持续动作里逼近答案。
- 内容任务：建立栏目第一条样片的表达范式，让观众知道这个账号不是人物传记，而是拆高手动作。
- 当前风险：容易被观众听成“宫崎骏很勤奋/很匠人”，而不是“宫崎骏有一套处理模糊问题的动作系统”。

### 第一轮：各角色独立判断

#### 主编 / 账号守门人

- 判断：账号方向是对的，但“我是谁、我为什么讲这个”的存在感还不够。
- 亮点：不是讲宫崎骏生平，而是明确拆一个高手动作。
- 问题：账号承诺放在最开头，会比对标视频更慢；第一条视频更需要先让观众被问题抓住。
- 必改：开头先抛“天才叙事让普通人放弃思考”，再补账号承诺。

#### 安先生结构审稿人

- 判断：主线清楚，但三证据结构还不够平衡。
- 亮点：“不是因为 A，而是因为 B”的翻案结构成立。
- 问题：分镜、手作是创作动作，AI 争议是价值判断，不适合并列为第三证据。
- 必改：第三证据改成“持续进入现场/生命经验来源”，AI 争议改为反向验证。

#### 舒老师文案审稿人

- 判断：痛点有了，但部分句子还偏概念。
- 亮点：“一团雾”“动作逼问答案”好记。
- 问题：“生命经验”“手作系统”这些词如果没有场景，容易变抽象。
- 必改：每个抽象词后面都补一个画面：手、纸、动作、停顿、修改、走路、回头。

#### 观众代表

- 判断：我愿意看，但我希望更快知道这和我有什么关系。
- 亮点：“天才这个词，最容易让普通人放弃思考”有记忆点。
- 问题：普通创作者的痛点在第 3 段才出现，稍晚。
- 必改：开头 30 秒内就要出现“你也有很多感觉，但它们还不是作品”。

#### 反对者 / 冷水评委

- 判断：现在最危险的是把宫崎骏说成“努力一点就行”。
- 亮点：稿件已经主动避免讲完整生平。
- 问题：如果没有纪录片时间码，“分镜是思考界面”会像漂亮解释。
- 必改：最终稿前必须补硬证据；结构稿里要明确哪些句子必须有画面或原话支撑。

#### 知识库承接官

- 判断：这条视频能自然沉淀成“高手动作卡”。
- 亮点：动作逼出答案，可以成为栏目母题。
- 问题：不要在视频里硬卖知识库；这条更适合建立信任。
- 必改：结尾只留下下一期期待，不做产品承接。

## 第二轮：争议点

### 争议 1：开头要不要先讲账号承诺？

- 分歧：对标视频先讲账号承诺，但我们的账号第一条视频如果也这么开，可能显慢。
- 为什么重要：新观众不是因为账号承诺留下，而是因为第一个认知缺口留下。
- 最终方向：先用“天才叙事”打认知缺口，再用一句话补账号承诺。

### 争议 2：AI 争议能不能做第三证据？

- 分歧：AI 争议有传播性，但它和分镜、手作不是同一层级。
- 为什么重要：三证据必须共同证明一个核心命题。AI 争议如果处理不好，会把视频带成技术站队。
- 最终方向：第三证据改为“生命经验来自持续进入现场”，AI 争议放在后面做反向验证。

### 争议 3：要不要继续强化普通创作者可迁移？

- 分歧：太早讲普通创作者，可能削弱人物质感；太晚讲，又会像人物欣赏。
- 为什么重要：这个账号不是传记号，必须让用户看到“我能借走一个动作”。
- 最终方向：开头 30 秒埋痛点，中段讲证据，结尾再收束为最小动作。

## 第三轮：最终裁决

- 结论：小修后可进入资料补强。
- 最大问题：第三证据错位，AI 争议不应和分镜、手作并列。
- 最该保留：“天才这个词，最容易让普通人放弃思考”和“动作逼出答案”。
- 必改项：
  - 开头顺序改为：认知缺口 → 普通创作者痛点 → 账号承诺。
  - 三证据改为：分镜、手作、现场/生命经验。
  - AI 争议改为反向验证，不做主证据。
  - 每个抽象判断后面补可拍画面。
- 可选优化：
  - 标题优先用“宫崎骏不是等灵感的人，他是用动作把答案逼出来的人”。
  - “反 AI”相关内容留给第二条，不要在第一条抢主线。
- 优先测试变量：
  - 观众是否会复述“动作逼出答案”，而不是只评论“宫崎骏真厉害”。
- 下一步：
  - 按下面修订版主结构，回改 `16_宫崎骏第一条视频结构稿.md`。
  - 再进入资料补强，补纪录片时间码和原话。

## 修订版主结构

建议把原结构从：

```text
账号承诺
→ 翻案式开头
→ 压低旧答案
→ 创作者处境
→ 分镜
→ 手作
→ AI 争议
→ 承认代价
→ 普通人可借走
→ 意象收束
```

改成：

```text
翻案式开头：宫崎骏不是靠想象力
→ 普通创作者痛点：你不是没有想法，而是想法还没被压成动作
→ 账号承诺：我们不是讲传记，而是拆高手动作
→ 压低旧答案：天才叙事太晚了
→ 核心判断：高手不是先想清楚再开始，而是在行动中逼近答案
→ 证据 1：分镜，把一团雾拆成一格一格的问题
→ 证据 2：手作，让创作者和材料形成最短反馈链路
→ 证据 3：持续进入现场，让动作背后有生命经验
→ 反向验证：AI 争议真正暴露的是“没有身体经验的动作”
→ 承认代价：动作可借，代价不必照搬
→ 普通人可借走：先做一个最小动作，让动作替你逼问答案
→ 意象收束：把一团雾一格一格逼成作品
```

## 需要回改到原稿的关键句

### 新开头方向

```text
宫崎骏之所以是宫崎骏，不是因为他想象力丰富。

这句话听起来像在故意反常识，但它真正要反的，不是宫崎骏的想象力，而是我们看高手时最偷懒的一种解释：把一切都归结为天才。

因为“天才”这个词，最容易让普通人放弃思考。

你一旦说他是天才，就不用再追问：他到底做了什么？他每天怎么处理那些还没成型的想法？他是怎么把脑子里的一团雾，变成一格一格能推进的作品？
```

### 第三证据新方向

```text
宫崎骏真正反复进入的，不只是画纸，而是现场。

他要观察一个人怎么走路，怎么回头，怎么停顿，怎么沉默。因为动画里最难的不是让人物动起来，而是让人物动得像真的活过。

所以他在意的不是风格像不像，而是这个动作背后有没有生活留下来的痕迹。
```

### AI 争议新位置

```text
这也是为什么，那段著名的 AI 争议不能被简单理解成“老人反技术”。

它更像一个反向验证：当一个动作只有形状，没有身体经验、痛感和生活记忆的时候，宫崎骏会本能地觉得它是空的。
```

### 结尾强化

```text
所以普通创作者能借走的，不是宫崎骏的画风，也不是他的性格。

你真正能借走的是一个动作：当你还没有想清楚的时候，别继续在脑子里等。写下一句话，画出第一格，列出三个场景，先做一个很粗糙的版本。

让动作替你判断，这个东西到底有没有生命。
```
