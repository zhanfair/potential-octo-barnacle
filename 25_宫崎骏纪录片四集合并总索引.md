## 总判断

四集过完后，第一条视频的主线不建议推翻。

仍然建议保留：

```text
宫崎骏不是靠等待灵感成为宫崎骏，
而是靠具体动作把模糊感受逼成作品。
```

但需要把副主线加强：

```text
真正的真实感，是生命经验进入动作。
```

也就是说，第一条视频不是单纯讲“宫崎骏很勤奋”，也不是单纯讲“宫崎骏反 AI”。

它应该讲：

> 他如何把模糊感受放进分镜、手作、现场和身体经验里，让作品一点点长出答案。

## 第一条视频最该使用的素材

### 主证据 1：分镜是思考界面

优先素材：

- 第一集 08:45-09:59：不是先有完整剧本，而是先从 image board 开始。
- 第一集 13:07-14:48：第一张图像让想法开始流动，但开场又被推翻。
- 第一集 42:54-43:25：亲手用铅笔画 storyboard。
- 第四集 15:51-16:08：分镜是电影制作基础，是电影蓝图。

用途：

证明分镜不是把答案画出来，而是逼问答案。

可用表达：

```text
一个人创作时，分镜是思考界面；
三百个人一起创作时，分镜就是行动地图。
```

### 主证据 2：手作是反馈链路

优先素材：

- 第一集 23:52-27:57：旧粉彩、线条、阴影、高光、用巨鱼表现海啸。
- 第二集 02:03-02:16：不用 CG，每张手画，五秒画面花一周。
- 第二集 04:40-06:42：检查每张图，不满意就自己重画，用脚和衣角表达波妞的喜悦。
- 第四集 40:06-41:37：病中的直子起身叠衬衫，宫崎骏要求动作尽可能真实。

用途：

证明手作不是怀旧，而是让情绪、材料和动作形成最短反馈。

可用表达：

```text
材料反馈的是画面能不能成立；
动作反馈的是情绪有没有落到身体上。
```

### 主证据 3：现场是生命记忆库

优先素材：

- 第一集 35:39-39:35：去濑户内海生活、散步、买菜、做饭、洗衣。
- 第二集 08:16-10:28：波妞来自真实孩子，三米半径的日常生活成为故事基础。
- 第二集 42:02-46:59：Toki 的站起与拥抱，接着宫崎骏自己的母亲记忆。
- 第四集 25:19-30:26：照片展让他重新理解“努力活着的人”，进而重画堀越的表情。

用途：

证明宫崎骏的想象力不是逃离现实，而是现实被看深以后长出幻想。

可用表达：

```text
真正的真实感，不是画得像。
是生命经验进入动作。
```

### 代价段主素材

优先素材：

- 第一集 15:14-15:27：长期在怀疑和焦虑中工作。
- 第三集 38:08-45:24：地震后制作危机，团队停工与复工，日夜工作。
- 第四集 20:07-21:38：年龄、身体和精神状态的代价。
- 第四集 42:00-44:43：还有 100 个分镜镜头要画，问题变成“怎样度过自己的日子”。

用途：

避免视频变成励志鸡汤。

可用表达：

```text
这套方法不温柔。
它要求你在没有答案时继续画，
在身体不舒服时继续画，
在时代压到你面前时继续画。
```

### 结尾主素材

优先素材：

- 第四集 40:06-41:37：病中的直子叠衬衫。
- 第四集 42:00-44:43：每一笔都像在问自己，怎样才算完整度过自己的日子。
- 第四集 44:43-46:42：宫崎骏少见地称赞自己的分镜。
- 第四集 48:14-48:18：两个月后宣布退休。

用途：

把结尾从“方法论收束”推到“生命状态收束”。

可用表达：

```text
作品不是生活之外的东西。
作品就是他回答自己怎么活的方式。
```

## 第一条视频不建议重用太多的素材

### 不建议重用第三集父子线

原因：

- 容易把视频带到宫崎骏与宫崎吾朗的父子关系。
- 会稀释第一条视频“高手动作拆解”的主线。
- 更适合后续单独做一条：`宫崎骏如何把创作压力传递给下一代导演`。

可保留：

- 分镜是导演最重要工作。
- 七万张手绘图需要导演统一。
- 地震后团队继续推进，作为代价段旁证。

### 不建议把 AI 争议提前

原因：

- 四集纪录片已经足够证明“身体经验进入动作”。
- AI 争议只适合作为反向验证，不适合作为主证据。

建议：

- 保留现在 `7.5 反向验证` 的位置。
- 不扩大。

## 对主稿的具体修改建议

### 1. 不大改前三个证据

前三个证据现在已经成立：

```text
分镜 = 思考界面
手作 = 反馈链路
现场 = 生命记忆库
```

不要继续往里面塞太多纪录片细节，否则会变成资料堆砌。

### 2. 升级代价段

当前代价段还偏抽象。

应该补三层具体代价：

1. 没有答案时继续画。
2. 身体和年龄不允许时继续画。
3. 时代、灾难、团队压力压上来时继续画。

### 3. 升级结尾段

当前结尾是：

```text
用铅笔、分镜、身体和时间，
把一团雾，一格一格逼成答案。
```

可以保留，但前面加一层：

```text
到最后，作品不是生活之外的东西。
作品就是他回答自己怎么活的方式。
```

### 4. 普通人可借走段不要写成坚持

普通创作者能借走的不是“坚持到底”，而是：

```text
把抽象情绪落到一个动作里。
把模糊判断落到一句话里。
把混乱想法落到一个结构里。
```

## 第一条视频最终素材配比建议

```text
第一集：45%
第二集：30%
第四集：20%
第三集：5%
```

原因：

- 第一集最贴合主命题：从 image board 开始，把模糊逼成作品。
- 第二集最贴合副命题：真实感来自生命经验进入动作。
- 第四集负责加重量：晚年、代价、没有答案时仍回到分镜。
- 第三集只做边缘补强：分镜是团队行动地图，代价不是个人情绪。

## 后续可拆出的第二条视频
