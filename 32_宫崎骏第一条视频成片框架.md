# 宫崎骏第一条视频成片框架

## 核心判断

可以按你说的方式做。

这条片子不需要每一句口播都精准找到纪录片对应画面。更稳的做法是：

```text
克隆声音口播
+ 纪录片核心证据片段
+ 工作场景 / 空镜 / 手部动作补位
+ 金色重点句字幕
+ 白色逐句字幕
= 一条可完成的字幕观点型人物拆解视频
```

也就是说，我们先搭一个完整成片框架，再慢慢替换局部画面。

## 成片形态

参考 `31_抖音花花参考视频类型拆解.md`，这条宫崎骏视频建议采用：

- 9:16 竖屏黑底。
- 中间放 16:9 纪录片横向画面。
- 上方保留小标题：`「宫崎骏的创作系统」`。
- 画面主体不追求高频切换，重点靠字幕推进。
- 每 15-25 秒出现一句金色大字。
- 白色字幕跟随口播逐句出现。
- 纪录片画面承担“可信度”和“情绪底色”，不是逐句解释。

## 2026-07-08 复审后调整

第一版竖屏字幕粗剪已经验证了“声音 + 纪录片画面 + 观点稿”的基本骨架，但当前主线先调整为：

- B 站横屏尺寸，优先保留纪录片原始横向画面。
- 暂时不生成字幕，只输出声音和画面，避免自动字幕切分破坏观感。
- 从完整纪录片里继续补充空镜和具体物象镜头，尤其是海、画稿、画画动作、工作台、生活现场。
- 对画面不贴的段落，不先改稿，优先替换素材。

也就是说，竖屏字幕版作为参考样片保留；下一轮制作以横屏无字幕版为当前主版本。

## 三层画面策略

### A 层：必须出现的证据画面

这些画面用于支撑核心论点，优先使用。

- 分镜 / image board
- 手作 / 线条 / 重画
- 现场生活 / 濑户内海
- Toki 拥抱 / 生命经验进入动作
- 年龄和身体代价
- 结尾的小动作

### B 层：补位画面

没有明确证据片段的段落，用这类画面补。

- 宫崎骏走进工作室。
- 桌面、铅笔、纸、分镜。
- 手部画画、翻页、看图。
- 工作室墙面、草图、团队工作。
- 海边、街道、生活空镜。
- 纪录片中的慢镜头、过渡镜头。

补位画面不需要完全对应台词，只要服务当段情绪。

### C 层：字幕画面

这条片子的观看重心在字幕。

- 金色大字：核心判断。
- 白色字幕：当前口播句。
- 小标题：栏目识别。
- 必要时用黑底金字卡，替代缺画面的段落。

## 粗剪规则

### 画面长度

- 每个镜头 3-8 秒。
- 证据画面可以稍长，但最好不要超过 12 秒连续使用。
- 补位画面可以重复，但要换裁切、推拉、明暗和字幕位置。

### 字幕节奏

- 白色字幕跟随口播。
- 每屏 10-22 字。
- 金色大字每 15-25 秒出现一次。
- 金色大字不要超过一行，最多两行。

### 画面优先级

优先级从高到低：

1. 台词正在讲具体证据，用对应纪录片片段。
2. 台词在抽象总结，用金色大字 + 工作场景。
3. 台词在讲普通创作者，用空白文档、笔记、结构稿这类自制画面。
4. 台词没有合适画面，用暗色纪录片空镜 + 大字幕推进。

## 第一版可以接受的“不完美”

第一版不需要做到每个镜头都完美。

允许：

- 某些段落只用工作室空镜补位。
- S02、S10、S12 先用字幕卡和通用工作画面。
- 同一个纪录片片段在不同段落重复，但裁切和字幕不同。
- 先做 8 分钟版本，后面根据观感再压稿。

不允许：

- 长时间无字幕。
- 一整分钟都只有同一个静态画面。
- 字幕挡住人脸、手部动作、分镜纸。
- 把纪录片片段整段搬上去，变成纪录片搬运。

## 下一步执行顺序

1. 按 `speed110` 分段音频建立时间线。
2. 每段先放一个主要纪录片片段。
3. 没有主要片段的段落，用工作场景/空镜/字幕卡补位。
4. 给每段加金色大字。
5. 再加白色逐句字幕。
6. 生成一个粗剪预览版。
7. 看完整体节奏后，再决定是否压稿或替换局部画面。

## 结论

这条路是可行的。

它会让我们从“还在找素材”进入“已经有一条完整视频骨架”的状态。后面所有优化都可以局部发生：某段不好，就换那段画面；某句拖，就删那句稿；某个字幕不好，就改那一屏。
