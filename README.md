# 高手思维拆解知识库商业体系

这是一套以“高手思维拆解”为起点的轻交付知识付费体系。第一阶段目标不是做大而全品牌，而是先验证现金流：用知识库降低交付压力，用社群文字答疑增强结果感，再把高意愿用户转化为“个人方法论诊断”。

## 当前定位

- 主产品：高手思维拆解知识库
- 目标用户：内容创业者、个人 IP、知识付费从业者
- 主入口：视频号
- 首发价格：599 元早鸟，699 元标准首发，跑通后 999 元
- 交付方式：飞书或 Notion 知识库 + 社群文字答疑
- 升级产品：个人方法论诊断，首批 999-1999 元，有案例后 2999 元以上

## 文件使用顺序

1. 先看 [00_商业作战图.md](./00_商业作战图.md)，确认整体生意结构。
2. 按 [01_知识库搭建蓝图.md](./01_知识库搭建蓝图.md) 搭建飞书或 Notion。
3. 用 [02_销售页文案.md](./02_销售页文案.md) 做首版销售页。
4. 用 [03_视频号内容预热计划.md](./03_视频号内容预热计划.md) 连续预热 15 条内容。
5. 按 [04_社群文字答疑规则.md](./04_社群文字答疑规则.md) 控制答疑边界。
6. 用 [05_方法论诊断升级产品.md](./05_方法论诊断升级产品.md) 承接高意愿用户。
7. 每周更新 [06_90天执行看板.md](./06_90天执行看板.md) 和 [07_指标追踪表.csv](./07_指标追踪表.csv)。

## 模板目录

`templates/` 里是可以直接复制到知识库的页面模板：

- 高手案例拆解模板
- 拆解方法卡模板
- 个人方法论画布
- 观点提炼卡
- 内容转化模板
- 用户案例沉淀卡
- 社群答疑沉淀卡

## 第一阶段验收标准

- 2 周内完成知识库 MVP。
- 4 周内发布 10-15 条视频号预热内容。
- 首月卖出 20 单知识库。
- 90 天卖出 50-100 单知识库，并成交 3-10 单个人方法论诊断。
